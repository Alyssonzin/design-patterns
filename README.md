# Design Patterns

Este repositório tem como objetivo documentar e exemplificar brevemente diversos padrões de desenvolvimento (Design Patterns), facilitando o entendimento e a consulta de implementações práticas.

## Como executar os exemplos

Para executar o código de exemplo de um padrão específico, utilize o comando configurado no `package.json`.

Exemplo:
```bash
npm run test -- <nomeDoPattern>/index
```

Para rodar o exemplo do padrão **Observer**:
```bash
npm run test -- observer/index
```
